## hazarddata.org

https://hazarddata.org

AVAILabs client for Hazard Data platform.

##### Installation

Create a `src/config.json` file using `src/config.json.template`.

```sh
$ git clone git@github.com:availabs/hazarddata.org.git
$ cd hazarddata.org
$ git submodule init
$ git submodule update
$ npm install
$ npm start
```
